# 🎯 QuizVerse

QuizVerse is a modern and interactive quiz platform built using **HTML, CSS, and Vanilla JavaScript**. It provides an engaging learning experience through multiple quiz categories, difficulty levels, real-time analytics, achievements, leaderboards, bookmarks, and progress tracking.

## ✨ Features

### 📚 Multiple Quiz Categories

* Technology
* Science
* Entertainment
* Sports
* General Knowledge
* Fun & Casual

### 🎮 Interactive Quiz Experience

* Multiple-choice questions
* Difficulty selection
* Question timer
* Progress tracking
* Skip questions
* Previous/Next navigation
* Real-time score calculation

### 📊 Analytics Dashboard

* Total quizzes attempted
* Average score
* Best score tracking
* XP and Level system
* Performance insights
* Learning streaks

### 🏆 Achievements & Rewards

* Unlock achievement badges
* Earn XP points
* Track progress
* Daily challenge mode

### 📈 Leaderboard

* Local leaderboard
* Best score tracking
* Recent quiz history

### 🔖 Bookmarks

* Save questions for later review
* Manage bookmarked content

### 🎨 Modern UI/UX

* Glassmorphism design
* Futuristic neon accents
* Responsive layout
* Animated backgrounds
* Smooth transitions
* Dark/Light mode support

## 🛠️ Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* Local Storage API

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/quizverse.git
```

2. Open the project folder

```bash
cd quizverse
```

3. Run the project

Simply open `index.html` in your browser.

No installation or external dependencies required.
